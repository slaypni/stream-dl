stream-dl
=========
Download streaming videos from websites.

Dependencies
------------
These programs should be on the PATH.
+ Node.js
+ FFmpeg

Install
-------
```
npm install -g stream-dl
```

Usage
-----
```
stream-dl [options] <url>
```

Example
-------
```
stream-dl -o video.mp4 http://example.com/path/to/watch/page
```

Build
-----
```
npm install
npm run build
```
